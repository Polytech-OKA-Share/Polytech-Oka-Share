## 自作の資料等を管理するアカウント

自作の訓練資料を管理します ^^  
著作権なんてありません。勝手にcloneして改変してください（コラボレーターはルールに則ってください ^^）  
Office系ファイル多いですが差分見たかったら tika とか入れてください。マクロ付きの文書は入れるつもりないので git-xl はやりすぎかもです。  

### ルールセット

コラボレーターは次のルールで作業してください。（**書きかけ！**）  
+ **branch構成・保護ルール**　mainは保護。GitHub-Flowで。
+ **branch名の規則**　JIRAの課題キーくらいは入れときたいね
+ **コミットメッセージ**　テンプレートを必ずリポジトリに入れときます。
+ **Pull Request**　これもテンプレート入れとく
+ **リンター・フォーマッター**　Pythonでしか使ってないけどJavaのもどうせあるでしょう。
+ **ディレクトリ構成**　インストール資材のリンク、教材、コードみたいな。これ無くてもいいかもしんない。
